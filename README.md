# Analog_Clock
🕒 Analog Clock

🚀 Description

Analog Clock is a simple and interactive web application that displays the current time using a real-time analog clock interface. It visually represents hours, minutes, and seconds with moving clock hands.

---

🎯 Features

- ⏱️ Real-time clock updates
- 🎨 Smooth rotating clock hands
- 📱 Responsive design
- ⚡ Lightweight and fast

---

🛠️ Tech Stack

- HTML
- CSS
- JavaScript

cd Analog_Clock

3. Run the project
   👉 Open "index.html" in your browser

---

💡 How It Works

- JavaScript gets current time using "Date()"
- Calculates rotation angles for:
  - Hour hand
  - Minute hand
  - Second hand
- Applies CSS transformations for movement
